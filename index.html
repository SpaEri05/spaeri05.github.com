About This Site

This site is a project for an Evergreen State College Internship through the Cooper Point Journal called Web Programming and Management. This Webpage will be where I record and display the progress made throughout the quarter. I will also be linking to external sites that we have used during this process. In the end, I will have a collection of all the work that I have done during this quarter and use this as a place to showcase my progress.

Objectives

These are the objectives intended for this quarter:

  • Learn to manage complex projects working with a team using open source software.
  • Learn to design the style and functionality of a web project.
  • Learn basic and intermediate programming, version control, linux/unix development, web framework and CMS development, and data visualization.
  • Learn how journalism can utilize data, data visualizations, application programming interfaces, web services, social media, and open source software/hardware.
